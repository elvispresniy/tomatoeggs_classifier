# Установка

После клонирования/установки репозитория скачать зависимости:

```bash
pip install -r requirements.txt
```

# Подготовка

Можно настроить конфигурацию в фалйе `config.py`.

Веса CNN-классификатора букв нужно поместить в папку `weights/` (предварительно создав её).

# Запуск

```bash
python app.py
```
